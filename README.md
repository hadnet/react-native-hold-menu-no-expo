A performant, easy to use hold to open context menu for React Native CLI projects powered by Reanimated.

## Installation

```sh
yarn add react-native-hold-menu-no-expo
```

## Usage

```js
import { HoldItem } from "react-native-hold-menu-no-expo";
```
